# Возможности

Пакет реализует типовые операции Telegram Bot API, необходимые для интеграции Telegram с Naumen Service Desk.

## Сообщения

- Отправка сообщения в чат.
- Редактирование сообщения в чате.
- Удаление сообщения.
- Отправка ответа на сообщение.
- Пересылка сообщения в другой чат.
- Постановка реакции на сообщение.
- Отправка локации.
- Отправка фотографии.

## Чаты и участники

- Получение администраторов чата.
- Получение участника чата.
- Получение списка участников по известным идентификаторам.
- Изменение названия чата.
- Изменение аватара чата.
- Блокировка и разблокировка пользователя.
- Выход бота из чата.

## Входящие события

- Работа с WebHook Telegram.
- Автоматическое обновление access key для WebHook.
- Получение событий через Long Polling.
- Общая обработка событий через `TelegramUpdateProcessor`.
